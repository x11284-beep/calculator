# Micro:bit Python Calculator

Simple calculator project made with Micro:bit and Python.

## Author

**Radim Sekac**

## Description

This is a simple calculator created with Micro:bit using Python.

The program allows the user to enter two numbers and display the result.

The numbers can be selected from 1 to 9.

## Controls

- **Button A** → Increase the current number (+1)
- **Button B** → Confirm the number / show the final result
